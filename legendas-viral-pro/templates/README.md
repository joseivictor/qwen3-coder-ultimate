# Modelos

Coloque seus modelos nesta estrutura:

- `1-linha/`
- `2-linhas/`
- `3-linhas/`
- `4-linhas/`
- `5-linhas/`

Formatos aceitos pelo painel:

- `.mogrt`
- `.cgt`
- `.cga`
- `.aep`

Para cada modelo, você pode adicionar:

- `thumb.jpg` ou `thumb.png` para miniatura.
- Um efeito sonoro `.wav`, `.mp3` ou `.aiff` na mesma pasta.

O painel agrupa os modelos pela pasta de linhas e permite escolher um modelo preferido para cada quantidade de linhas.

Além dos modelos locais, o painel também tenta ler automaticamente a biblioteca instalada em:

```text
C:\Program Files (x86)\Common Files\Adobe\CEP\extensions\Legendas Master 3.7\templates
```

Essa leitura é somente de uso. O plugin novo não edita a biblioteca original.
