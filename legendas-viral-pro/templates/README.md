# Modelos

Coloque seus modelos nesta estrutura:

- `1-linha/`
- `2-linhas/`
- `3-linhas/`
- `4-linhas/`
- `5-linhas/`

Formatos aceitos pelo painel:

- `.mogrt`
- `.cgt`
- `.cga`
- `.aep`

Para cada modelo, vocÃª pode adicionar:

- `thumb.jpg` ou `thumb.png` para miniatura.
- Um efeito sonoro `.wav`, `.mp3` ou `.aiff` na mesma pasta.

O painel agrupa os modelos pela pasta de linhas e permite escolher um modelo preferido para cada quantidade de linhas.
