# Reference Sites

## Primary extraction targets

- `awwwards_home` — https://www.awwwards.com/
  - Use for premium structure, hierarchy, editorial composition, examples of high-end web direction.
- `dribbble` — https://dribbble.com/
  - Use for card design, visual polish, product UI patterns, color/mood exploration.
- `dark_design` — https://www.dark.design/
  - Use for dark premium atmosphere, black/amber contrast, visual restraint.
- `shadergradient` — https://shadergradient.co/
  - Use only for shader/gradient atmosphere, not rustic commerce.
- `moncy` — https://www.moncy.dev/
  - Use for portfolio-grade motion, creative transitions, immersive sections.
- `naresh_khatri` — https://www.nareshkhatri.site/#projects
  - Use for portfolio structure, project cards, section rhythm, animated navigation.
- `portfolio_3d` — https://3d-portfolio-website-gamma.vercel.app/
  - Use for 3D hero object ideas, depth, floating product treatment.
- `durves` — https://www.durves.com/
  - Use for bold interactive layout and unconventional transitions.
- `calltoinspiration` — https://calltoinspiration.com/
  - Use for microinteraction ideas, buttons, menus, hover states.
- `omma_build` — https://omma.build/
  - Use for premium 3D/interactivity, product-like hero, cinematic scroll composition.

## Recommended combinations

### Premium dark shop
Use `dark_design`, `awwwards_home`, `dribbble`, `omma_build`.

### Immersive portfolio / motion site
Use `moncy`, `portfolio_3d`, `naresh_khatri`, `durves`, `shadergradient`.

### Product/e-commerce with visual taste
Use `awwwards_home`, `dribbble`, `omma_build`, `calltoinspiration`.

### Rustic luxury commerce
Use `dark_design` for contrast, `awwwards_home` for hierarchy, `dribbble` for product cards, then add original rustic material direction. Do not overuse abstract shader references.
